# Accident AI — Emergency Response System

Accident AI is a cutting-edge, AI-powered emergency management platform designed to minimize response times during critical incidents. By leveraging advanced sensors and machine learning, it provides real-time accident detection and autonomous rescue orchestration.

## 🚀 Key Features

- **Autonomous SOS**: AI-driven impact detection with a fail-safe countdown mechanism.
- **Biometric Profile**: Securely stores critical medical information (Blood Type, Allergies) for first responders.
- **Rescue Orchestration**: Real-time coordination between victims, emergency services, and hospitals.
- **Multi-lingual Support**: Integrated Google Translate support for 10+ Indian languages to ensure accessibility.
- **Intelligent Analysis**: Powered by Gemini 1.5 Pro for generating high-precision emergency plans.

## 🛠️ Tech Stack

- **Frontend**: React + Vite + TypeScript
- **Styling**: Tailwind CSS + Framer Motion (for fluid, premium animations)
- **UI Components**: Shadcn UI + Radix Primatives
- **AI Engine**: Google Gemini 1.5 Pro
- **Deployment**: Vercel

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- npm or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ahamedriyaz-5/accident-detection-ai.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## 🏗️ Project Structure

- `src/components`: UI screens and shared components.
- `src/services`: AI logic and external service integrations.
- `src/contexts`: Global state management for user profiles.
- `src/lib`: Utility functions and helper classes.
